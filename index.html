BWLink - URL shortener/redirector

API Endpoints

This service also provides API endpoints for programmatic access:

Shorten URL API

POST /shorten
Content-Type: application/json

{
  "password": "your_api_password",
  "longURL": "https://example.com/long/url"
}

Access Shortened URL

GET /redirect/shortCode
(automatically redirects to original URL with parameters preserved)

Note: The API endpoints (/shorten and /redirect) are configurable via environment variables.
